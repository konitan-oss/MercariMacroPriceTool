### 実行手順・コマンド
- 環境変数に dotnet パスを追加: `$env:PATH='C:\\Program Files\\dotnet;' + $env:PATH`
- ソリューション作成: `dotnet new sln -n MercariMacroPriceTool`
- プロジェクト作成: `dotnet new wpf -n MercariMacroPriceTool.App -o src/MercariMacroPriceTool.App` 他クラスライブラリ3件
- 参照追加: `dotnet add src/MercariMacroPriceTool.App/... reference ...` など
- ビルド: `dotnet build MercariMacroPriceTool.sln`
- 起動確認（5秒後に停止）: `dotnet run --project src/MercariMacroPriceTool.App/MercariMacroPriceTool.App.csproj --no-build` を `Start-Process` で起動し5秒後に終了
